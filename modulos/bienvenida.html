Bienvenido a easySQL-server

Esta aplicación te permite gestionar bases de datos en Supabase de forma visual y segura.
Flujo de uso:

  1. Sube tu archivo .easySQL (con credenciales y configuración encriptada) o crea una nueva llave si es tu primer uso.
  2. Accede con tu usuario y contraseña. El menú se adaptará a tu rol (dev o admin).
  3. Como dev puedes crear tablas, enumerados y realizar inserciones. Como admin solo puedes hacer inserciones.
  4. Los tipos enumerados (ENUM) pueden ser creados y usados en tablas e inserciones.
  5. La navegación está protegida para evitar errores y garantizar la seguridad de tus datos.
Recuerda: Para pruebas locales, usa python3 -m http.server 8000 y accede a http://localhost:8000.
